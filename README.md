# JMeter

- Download from https://jmeter.apache.org/

# Plugins

- Download plugins-manager.jar from https://jmeter-plugins.org/
- Put plugins-manager.jar into the `lib/ext` directory
- Restart JMeter
- Use the UI: Options > Plugins Manager
